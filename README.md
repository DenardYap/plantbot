# PlantBot

An AI agent that takes real care of real plants — wired to a Raspberry Pi with
sensors, a pump, and a live camera. This repo is the **website**: the place
where you pick a plant, watch its camera, and chat with it.

## Getting started

```bash
npm install
cp .env.example .env.local   # fill in later — defaults are fine for dev
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) — you'll be redirected to
`/plants`.
